Some projects

Waiter Calling System (frontend)

This application can be used as an electronic panel in cafeterias to show customers when their order is ready.

The project was developed with the aim of demonstrating the skills acquired in frontend development.

In this application I used typescript with the angular framework, socket.io client to exchange messages in real time and angular material as a library of visual components.

Waiter Calling System (backend)

This application can be used as an electronic panel in cafeterias to show customers when their order is ready.

The project was developed with the aim of demonstrating the skills acquired in backend development.

For this system module I used javascipt being interpreted by node.js, express as the base framework and some packages such as dotenv to store constants and sensitive information, express-validator to check the data coming from the frontend, json-server to simulate an API and a database and socket.io for exchanging messages in real time.

CRUD (backend)

CRUD of users, products and orders.
